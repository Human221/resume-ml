# Описание решения: HR-агент для анализа вакансий

## Краткое описание

Бизнес-ориентированный AI-агент для автоматизации работы HR-отделов с вакансиями. Решение использует MCP-протокол для интеграции с данными о вакансиях и Evolution Foundation Models для обработки естественного языка.

## Бизнес-ценность

### Проблема
HR-менеджеры тратят много времени на:
- Поиск подходящих вакансий по различным критериям
- Анализ рынка труда и зарплатных трендов
- Сравнение предложений от разных работодателей

### Решение
AI-агент автоматизирует эти задачи, позволяя:
- **Экономить 2-3 часа в день** на рутинных операциях
- **Улучшать качество найма** через быстрый анализ рынка
- **Масштабировать процессы** для компаний любого размера

## Техническая реализация

### Компоненты

1. **MCP-сервер** (`mcp_server/vacancies_mcp.py`)
   - 3 инструмента: поиск, статистика, сравнение
   - Валидация через Pydantic
   - Обработка ошибок и таймаутов
   - Работа с CSV данными

2. **AI-агент** (`agent/hr_vacancies_agent.py`)
   - Интеграция с Evolution Foundation Models
   - Обработка естественного языка
   - Вызов MCP-инструментов
   - Форматирование ответов A2A

### Инструменты MCP

1. **search_vacancies** - Поиск вакансий
   - Параметры: query, min_salary, max_salary, experience, area, limit
   - Возвращает список подходящих вакансий

2. **get_vacancy_statistics** - Статистика
   - Параметры: role, area, specialization (опционально)
   - Возвращает: количество, средняя зарплата, топ ролей/регионов

3. **compare_vacancies** - Сравнение
   - Параметры: vacancy_ids (2-5 вакансий)
   - Возвращает детальное сравнение

## Использование

### Установка
```bash
pip install -r requirements.txt
cp .env.example .env
# Отредактируйте .env и укажите EVOLUTION_API_KEY
```

### Запуск
```bash
# Демонстрация
python demo.py

# Интерактивный режим
python run_agent.py

# Тесты
python test_mcp.py
python test_agent.py
```
