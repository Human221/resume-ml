# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Настройка конфигурации

Скопируйте `.env.example` в `.env` и укажите ваш API ключ:

```bash
cp .env.example .env
```

Отредактируйте `.env`:
```
EVOLUTION_API_KEY=ваш_api_ключ_здесь
```

## 3. Проверка MCP-сервера

```bash
python test_mcp.py
```

## 4. Запуск демонстрации

```bash
python demo.py
```

## 5. Интерактивный режим

```bash
python run_agent.py
```

## Примеры запросов

- "Найди вакансии Python разработчика"
- "Покажи статистику по вакансиям программиста"
- "Найди вакансии с зарплатой от 150000 до 250000 рублей в Москве"
- "Сравни вакансии с ID 12345 и 67890"

## Требования

- Python 3.9+
- Evolution Foundation Models API ключ
- CSV файл с вакансиями (опционально, если нет - будет работать с пустыми данными)
